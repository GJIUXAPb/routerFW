findsni.sh v0.2
Linux Утилита для автоматического поиска и проверки корректных SNI для вашего домена.

Задача:
У вас есть ip адрес сервера, который находится в подсети с другими адресами под которые мы можем мимикрировать Reality.
Данный скрипт позволяет найти корректные конткретные и рабочие для вашего сервера SNI(tls1.3)

Описание:
RealiTLScanner-linux-64 - Open Source утилита которая находит такие домены. https://github.com/XTLS/RealiTLScanner/
Далее с помощью bash команды мы фильтруем её выход и следим по таймеру чтобы она не работа вечно.
Всё что она нашла за 2 минуты (время настраивается в коде скрипта) мы пишем в файл tocheck.txt
Далее проверяем этот файл ещё одной утилитой, которая следит чтобы сайт работал и давал верный код ответа с TSL1.3

Использование:
chmod +x findsni.sh && chmod +x RealiTLScanner-linux-64
./findsni.sh <IP-адрес вашего хоста>
